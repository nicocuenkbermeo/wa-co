7 reglas de oro no-violables.

  1. prefers-reduced-motion primero. Cada animación tiene su escape a estático.
  2. SSR-safe. Si JS no carga, el contenido se ve. Ningún initial:opacity:0 sin fallback.
  3. Performance antes que wow. LCP ≤ 2.5s, CLS 0. Lighthouse 90+ mobile siempre.
  4. Mobile igual de importante. No "desktop-only magic". Responsive real.
  5. Un efecto por sección. Ratio 1:1. No saturar — cada animación sirve al contenido, no al reel.
  6. Accent color matching. Cada primitivo acepta prop color, respeta el brand del proyecto.
  7. Cero glass-morphism genérico. Es 2022 design. Cream + selva + italic es 2026.
